# Chat App Example

Real-time chat room using WebSockets.

## Features
- WebSocket connections with auto-reconnect
- Session-based username
- Broadcast messaging to all connected clients
- Join/leave notifications
- Clean UI with tina4-css

## Run
```bash
cd examples/chat
python app.py
# Open http://localhost:7148
# Open a second browser tab and join with a different name
```
